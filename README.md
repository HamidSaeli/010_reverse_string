# Project 010: Reverse a String

## Description
This project asks the user to enter a string and then prints the reversed version of that string.  
It teaches beginners how string slicing works in Python, especially the `[::-1]` trick.

## Skills Learned
- String slicing  
- User input  
- Working with strings  
- Basic printing  

## How to Run
1. Open `reverse_string.ipynb` in Jupyter Notebook.
2. Run all cells.
3. Enter any text and see its reversed form.
